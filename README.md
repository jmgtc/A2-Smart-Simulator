# A2 Smart Simulator PRO v2.0

Simulador de examen para el carnet de moto A2 basado en la normativa DGT 2026.

## Características
- 🔄 **Bimodal**: Audio y Voz interactiva.
- 🧠 **Smart Review**: Algoritmo de refuerzo para fallos.
- 🎨 **Estética Premium**: Diseño Glassmorphism.
- 📱 **PWA**: Instalable como aplicación móvil.
